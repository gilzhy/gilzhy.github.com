Android学习笔记-基础知识

说明: 内容取自中山大学软件学院刘宁老师的SE-806课程的课件

几个概念

  • ADT: Android Development Tools, Eclipse的插件
  • AVD: Android Virtual Devices, Android虚拟设备
  • Android Emulator: Android模拟器, QEMU
  • Dalvik: Android所用的虚拟机,可以支持.dex格式程序的运行
  • .dex: Dalvik EXecutable, 转为Dalvik设计的一种压缩格式,适合内存和处理器速度有线的系统
  • dx: java byte code to Dalvik executable translator,将class文件转为dex文件工具
  • .apk: Android安装包的扩展名,包含应用相关所有文件,本质是一个zip包
  • Two Debuggers:
    • adb: Android Debug Bridge, Android调试桥
    • ddms: Dalvik Debug Monitor Server, 调试监视服务
  • aapk: Android Application PacKage Tool, 资源打包工具
  • 创建项目时:
    • Project Name: 计算机中存储的项目的目录的名字
    • Application Name: 显示在模拟器上的应用的名字
    • Package Name: 包名,同java
    • Activity Name: UI界面窗口的名字
    • Min SDK Version: 最低api说明

文件类型转换过程

.java —编译—> .class —dx工具—> .dex —aapk工具—> .apk

应用的文件组成

  • R.java: 建立项目时自动生成,只读模式, 对将要用到的资源进行全局索引,Res文件夹中发生任何变化, R.java都会重新编译
  • Android Library: Android库
  • asserts目录: 多媒体等文件
  • res目录: 应用用到的资源文件
    • drawable目录: 图片资源
    • layout目录: 布局文件
    • values目录: string.xml, color.xml, array.xml
  • AndroidManifest.xml: 相当于应用的配置文件,声明应用名称,程序入口点, Acticity, Service 和receiver等

不用eclipse命令行操作

进入sdk的tools目录

  • android: 启动Android SDK and AVD manager
  • emulator -avd XXX: 启动模拟器
  • android list targets: 列出所有的SDK可用版本
  • adb install XXX.apk: 安装apk文件,如果有多个设备,而想安装到一个特定的模拟器,则输入:adb install -s emulator-5554 /home/zhyin/xxx.apk

模拟器屏幕

  • HVGA-L:480x320横屏
  • HVGA-P:320x480竖屏(默认)
  • QVGA-L:320x240横屏
  • QVGA-P:240x320竖屏
  • WVGA-L:800x480横屏

Neural Network

Neural Network

多分类输出

以四分类为例:

\[\begin{bmatrix}1\\0\\0\\0\end{bmatrix},\begin{bmatrix}0\\1\\0\\0\end{bmatrix},\begin{bmatrix}0\\0\\1\\0\end{bmatrix},\begin{bmatrix}0\\0\\0\\1\end{bmatrix}\]

分别表示样本预测结果为第1,2,3,4种分类

损失函数

Logistic regression的损失函数:

\[J(\theta) = - \frac {1}{m} [\sum\limits_{i=1}^{m}y^{(i)}\log h_{\theta}(x^{(i)})+(1-y^{(i)})\log (1-h_{\theta}(x^{(i)}))] + \frac {\lambda}{2m} \sum\limits_{j=1}^{n}\theta_{j}^{2}\]

Neural Network的损失函数:

\[J(\theta) = - \frac {1}{m} [\sum\limits_{i=1}^{m}\sum\limits_{k=1}^{K}y_k^{(i)}\log (h_{\theta}(x^{(i)})_k)+(1-y_k^{(i)})\log (1-h_{\theta}(x^{(i)})_k)] + \frac {\lambda}{2m} \sum\limits_{l=1}^{L-1}\sum\limits_{i=1}^{s_l}\sum\limits_{j=1}^{s_{l+1}}(\theta_{ji}^{(l)})^2\]

Neural Network的损失函数与Logistic regression的只基本一直, 只因为Neural Network的输出是一个K维向量(K为分类的类别数), 所以要把K维的每一维都计算一遍.

数据

m个样本, 每个样本n维

1. 关于 \(\mathbf{z}, \mathbf{a}\)

L层神经网络(Layer 1, Layer 2, …, Layer L), 每层有 \(s_l\) 维(不包括增加的1)

  • 第一层为输入层 \(\mathbf{a}^{(1)}=\mathbf{x}\) , x为其中一个样本(列向量)
  • 最后一层为输出层, 若分类问题一共有K中分类,则最后一层有K维, \(s_L=K\)

对于第i层:

  • \(\mathbf{z}^{(i)}=\mathbf{\theta}^{(i-1)}\mathbf{a}^{(i-1)}\)
  • \(\mathbf{a}^{(i)}=g(\mathbf{z}^{(i)})\), \(g(\mathbf{z})=\frac {1}{1+e^{-\mathbf{z}}}\)
  • 第i层共有 \(s_i\) 个值, 其中第j个值为: \(\mathbf{z}^{(i)}_j=\mathbf{\theta}^{(i-1)}_{j}\mathbf{a}^{(i-1)}\)

\(\mathbf{z}\) 和 \(\mathbf{a}\) 的值通过正向传播得到.

2. 关于\(\theta\)

\(\theta\) 记录神经网络中每一层每个结点的权重值.

\(\theta^{(l)}\) 表示第l层 \(s_l\) 个结点产生第 l+1 层 \(s_{l+1}\) 个结点的 \(s_l \times s_{l+1}\) 个权重.

因为 \(\mathbf{z}^{(i)}_j = \mathbf{\theta}^{(i-1)}_{j}\mathbf{a}^{(i-1)}\) , 所以有:

  • \(\mathbf{z}^{(i)}\) 中的j有几种取值 \(\theta^{(i-1)}\) 的j就有几种取值, 故 \(\mathbf{\theta}^{(i-1)}\) 中 \(s_i\) 个列向量
  • \(\mathbf{\theta}^{(i-1)}_{j}\) 和 \(\mathbf{a}^{(i-1)}_{j}\) 维度相同(a要先增加一维1), 而 \(\mathbf{a}^{(i-1)}_{j}\) 增加一维后的维度为 \(s_{i-1}+1\) , 故 \(\mathbf{\theta}^{(i-1)}\) 中每个向量的维度为 \(s_{i-1}+1\)

所以 \(\theta^{i-1}: (s_{i-1} + 1) \times s_{i}, \theta^{(i-1)}=\begin{bmatrix}\theta^{(i-1)}_1 & \theta^{(i-1)}_2 & ... & \theta^{(i-1)}_{s_{i}}\end{bmatrix}\)

\(\theta\) 的值是算法训练的目标.

3. 关于\(\delta\)

\(\delta\) 记录神经网络中每个结点的误差值(它为最后的预测值负多大责任). \(\delta^{(l)}_{j}\) 表示第l层第j个结点的误差值.

  • 第一层不需要计算 \(\delta\) , 因为Layer 1是输入层, 没有误差.
  • 最后一层的 \(\delta\) 可以很容易得到: \(\delta^{(L)}=a^{(L)}-y\)

而其他的hidden层需要通过反向传播(Back propagation)得到.

反向传播算法:

  • set \(\Delta^{(l)}_{ij}=0\) (for all l, i, j, 作为累加器来计算 \(\frac {\partial}{\partial\theta^{(l)}_{ij}}J(\theta)\) )
  • set \(\theta^{(l)}_{ij}=randrange(-\epsilon,+\epsilon)\) (for all l, i, j)
  • for i = 1 to m:
    • set \(a^{(1)}=x^{(i)}\)
    • perform forward propagation to compute \(a^{(l)}\) for l = 2, 3, …, L
    • 使用 \(y^{(i)}\) 计算 \(\delta^{(L)}=a^{(L)}-y^{(i)}\)
    • 使用 \(\delta^{(i)}=(\theta^{(i)})^T\delta^{(i+1)}.*g'(z^{(i)})\) 计算 \(\delta^{(L-1)}, \delta^{(L-2)}, ..., \delta^{(2)}\) (没有 \(\delta^{(1)}\) )
    • \(\Delta^{(l)}_{ij} := \Delta^{(l)}_{ij} + a^{(l)}_{j}\delta^{(l+1)}_{i}\) 或向量的方式计算: \(\Delta^{(l)}:=\Delta^{(l)}+\delta^{(l+1)}(a^{(l)})^T\)
  • \(D^{(l)}_{ij}:=\frac {1}{m}\Delta^{(l)}_{ij}+\lambda\theta^{(l)}_{ij}\ if\ j \neq 0\)
  • \(D^{(l)}_{ij}:=\frac {1}{m}\Delta^{(l)}_{ij}\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ if\ j = 0\)
  • \(\frac {\partial}{\partial\theta^{(l)}_{ij}}J(\theta)=D^{(l)}_{ij}\)

公式 \(\delta^{(i)}=(\theta^{(i)})^T\delta^{(i+1)}.*g'(z^{(i)})\) 的解释:

举例说明: 2维样本, 2分类问题, 神经网络有输入层, 输出层, 以及两个2维的hidden层. \(s_1=2,s_2=2,s_3=2,s_4=2\)

首先通过正向传播得到 \(\mathbf{z}\) 和 \(\mathbf{a}\) :

Layer 1:

\[\begin{matrix}\mathbf{a}^{(1)}_{1} = \mathbf{x}^{(i)}_{1}\\ \mathbf{a}^{(1)}_{2} = \mathbf{x}^{(i)}_{2}\end{matrix}\]

Layer 2:

\[\begin{matrix}\mathbf{z}^{(2)}_{1}=\theta^{(1)}_{1}\mathbf{a}^{(1)} &\mathbf{a}^{(2)}_{1}=sigmoid(\mathbf{z}^{(2)}_{1})\\ \mathbf{z}^{(2)}_{2}=\theta^{(1)}_{2}\mathbf{a}^{(1)}&\mathbf{a}^{(2)}_{2}=sigmoid(\mathbf{z}^{(2)}_{2})\end{matrix}\]

Layer 3:

\[\begin{matrix}\mathbf{z}^{(3)}_{1}=\theta^{(2)}_{1}\mathbf{a}^{(2)} &\mathbf{a}^{(3)}_{1}=sigmoid(\mathbf{z}^{(2)}_{1})\\ \mathbf{z}^{(3)}_{2}=\theta^{(2)}_{2}\mathbf{a}^{(2)}&\mathbf{a}^{(3)}_{2}=sigmoid(\mathbf{z}^{(2)}_{2})\end{matrix}\]

Layer 4:

\[\begin{matrix}\mathbf{z}^{(4)}_{1}=\theta^{(3)}_{1}\mathbf{a}^{(3)} &\mathbf{a}^{(4)}_{1}=sigmoid(\mathbf{z}^{(3)}_{1})\end{matrix}\]

然后反向传播计算 \(\delta\)

Layer 4:

\[\delta^{(4)}_1=y^{(i)}-a^{(4)}_1\]

Layer 3:

第1个结点的误差: \(\delta^{(3)}_{1}\) , 其误差大小影响到Layer 4的第1个结点的误差 \(\delta^{(4)}_{1}\) , 影响权重为 \(\theta^{(3)}_1\) , 故

\[\delta^{(3)}_{1} = \theta^{(3)}_1 \times \delta^{(4)}_{1}\]

第2个结点的误差: \(\delta^{(3)}_{2}\) , 其误差大小影响到Layer 4的第1个结点的误差 \(\delta^{(4)}_{1}\) , 影响权重为 \(\theta^{(3)}_2\) , 故

\[\delta^{(3)}_{2} = \theta^{(3)}_2 \times \delta^{(4)}_{1}\]

Layer 2:

第1个结点的误差: \(\delta^{(2)}_{1}\) , 其误差大小影响到第3层第1第2个结点的误差 \(\delta^{(3)}_{1}\)和\(\delta^{(3)}_{2}\) , 影响权重分别为 \(\theta^{(2)}_{11}\) 和 \(\theta^{(2)}_{21}\) , 故

\[\delta^{(2)}_{1}=\theta^{(2)}_{11} \times \delta^{(3)}_{1} + \theta^{(2)}_{21} \times \delta^{(3)}_{2}\]

第2个结点的误差: \(\delta^{(2)}_{2}\) , 其误差大小影响到第3层第1第2个结点的误差 \(\delta^{(3)}_{1}\) 和 \(\delta^{(3)}_{2}\) , 影响权重分别为 \(\theta^{(2)}_{12}\) 和 \(\theta^{(2)}_{22}\) , 故

\[\delta^{(2)}_{2}=\theta^{(2)}_{12} \times \delta^{(3)}_{1} + \theta^{(2)}_{22}\times \delta^{(3)}_{2}\]

直观规律总结: 对第i层第j个结点来说, 其误差 \(\delta^{(i)}_j\) 等于其误差大小影响到的第i+1层的结点的误差的权重和.

公式: \(\delta^{(i)}=(\theta^{(i)})^T\delta^{(i+1)}.*g'(z^{(i)})\) 其中 \(g'(z^{(i)})=a^{(i)}.*(1-a^{(i)})\) :
